Mini moteur de jeu & Demonstrateur du Tetris.
version 2.0
##Auteurs

##Date

##Introduction
Le programme est un demonstrateur du projet Tetris.
##Details
* Projet : Tetris
* Langage : C++11/ Qt
* IDE : QtCreator
* Source Control : GIT (https://github.com)
* Documentation : Doxygen
* ModéliseurUML : Visio & Umbrello

    * tester la chute
    * tester la fin du jeu
* Description :
    * Carte : matrice
